# vaniTV

A modern IPTV addon for Stremio.

## Features

- IPTV (M3U)
- Live TV
- Fast loading
- Open Source

## Requirements

- Node.js 22+
- npm

## Install

```bash
npm install
```

## Run

```bash
node src/index.js
```

Addon:

http://127.0.0.1:7000/manifest.json
